# Personal Portfolio Website

A modern, responsive portfolio website with a fixed footer and dynamic content loading. Features a contact form that stores submissions in PostgreSQL database.

## 🌟 Features

- **Dynamic Content Loading**: Smooth page transitions without full page reloads
- **Fixed Footer**: Footer stays at bottom while content scrolls
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Contact Form**: AJAX-based form submission with PostgreSQL storage
- **Admin Dashboard**: View and manage all contact form submissions
- **Modern UI**: Gradient backgrounds, smooth animations, and clean design
- **Mobile Menu**: Hamburger menu for mobile devices
- **Browser History**: Back/forward buttons work with content navigation

## 🛠 Technologies Used

### Backend
- **Node.js** - JavaScript runtime
- **Express.js** - Web application framework
- **PostgreSQL** - Relational database
- **Sequelize** - ORM for database operations
- **dotenv** - Environment variable management

### Frontend
- **EJS** - Template engine
- **HTML5** - Structure
- **CSS3** - Styling with Flexbox/Grid
- **Vanilla JavaScript** - Dynamic content loading and form handling

### Development Tools
- **Nodemon** - Auto-restart during development
- **Git** - Version control

## 📦 Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js** (v14 or higher) - [Download](https://nodejs.org/)
- **npm** (v6 or higher) - Comes with Node.js
- **PostgreSQL** (v12 or higher) - [Download](https://www.postgresql.org/download/)
